Study Sync helps you stay consistent with your exam prep through daily study plans and progress tracking.

🤖 Study Sync — Command Guide

/start — Bot setup kijiye (language, naam, exam, syllabus)
/progress — Subject-wise progress aur apni streak dekhiye
/pdf (ya /extractquestions) — Kisi PDF se MCQ questions nikaal kar clean, formatted PDF banwaiye
/addreminder HH:MM — Din mein kabhi bhi ek naya task reminder add kijiye
/removereminder HH:MM — Koi reminder hataiye
/myreminders — Apne saare active reminders dekhiye
/help — Yeh guide dobara dekhiye

📌 Bot kaise kaam karta hai:
• Roz subah aapko syllabus se ek study plan milega, aur shaam ko uska checklist
• Chahen to /addreminder se din mein aur reminders set kijiye — har ek par ek naya topic milega, turant tick kar sakte hain
• Jab chahe /progress se dekhiye kitna syllabus cover ho chuka hai
• Purane papers/notes ki PDF bhej kar usse questions alag nikalwa sakte hain
